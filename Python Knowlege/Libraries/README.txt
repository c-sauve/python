Some example Librarys

- math

- SciPy is an ecosystem of Python, distinct and separately maintained packages for math, science, and engineering. This ecosystem is composed of:

  -  NumPy: array package -> https://numpy.org/
  -  SciPy library: scientific computing package -> https://scipy.org/scipylib/index.html
  -  matplotlib: 2-d plotting package -> https://matplotlib.org/
  -  Ipython: interactive console package, Jupyter Notebook -> https://ipython.org/
  -  SymPy: symbolic math package -> https://www.sympy.org/en/index.html
  -  pandas: data structure and analysis package -> https://pandas.pydata.org/
